## Tipos de sintaxe
- O sass possui duas sintaxes (SASS) e (SCSS)

### Sintaxe SASS
- Não utiliza { } nem ; na estilização
#### Exemplo:
* seletor
    * propriedade: valor

### Sintaxe SCSS
- A sintaxe SCSS é igual do CSS com { } e ;
- É a sintaxe mais convencional
